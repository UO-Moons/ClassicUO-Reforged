# Introduction

ClassicUO Reforged is a custom Ultima Online client project focused on expanded client functionality, improved visuals, enhanced atmospheric effects, tooling support, and ongoing engine improvements.

The project is based on the ClassicUO codebase and continues development independently.

The client is written in C# using the FNA framework.

![screenshot_2020-07-06_12-29-02](https://user-images.githubusercontent.com/20810422/208747312-04f6782f-3dc8-4951-b0a0-73d2305bbfca.png)

# Platform Support

ClassicUO Reforged currently supports:

* Windows [DirectX 11, OpenGL, Vulkan]
* Linux   [OpenGL, Vulkan]
* macOS   [Metal, OpenGL, MoltenVK]

# Building

```bash
git clone https://github.com/UO-Moons/ClassicUO-Reforged.git
cd ClassicUO-Reforged/scripts
bash build-naot.sh
```

Binaries will be available in:

```text
bin/dist
```

> [!WARNING]
> To execute .sh scripts on Windows, use Git Bash:
> https://git-scm.com/download/win
